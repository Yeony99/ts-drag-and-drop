# ts-drag-and-drop
Practice for using TS

## How to Run

### Install `node_modules`

```bash
npm install
```

### Compile

```bash
tsc src/app.ts
```

### Run

```bash
npm run start
```

Then you can see the project on http://localhost:3000/ 😁
